Note: information on this page refers to Ceylon 1.1, not to the current release.

package

package can be used to qualify a reference.

Usage 

void example() {}

class Example() {
    void example() {
        // Calls the top level example, does not recurse.
        package.example();
    }
}

Description

package itself is not an expression but is permitted to qualify an expression.

Type

Because package itself is not an expression it does not have a type.

If you want to obtain an instance representing a package you can use a declaration reference: `package com.example`.
